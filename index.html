Git Fundamentals

Step#1 Git Repository

Follow these steps :

  • Log in your github account at any brower.
  • To make a git repository, do as shown in the picture:
  • Click on "+" sign and click "New repository"

This is how we can add a git repository to our account...

Step#2 Make a Git Repository

Follow these steps to make a git repository :

  • Enter repository name
  • Click on create repository button

We can either make it as public repository (will be visibel to everyone) or a private repository (will be visible to only you or any collaborators you add).

Step#4 Make a Project on your localhost/Personal Computer on VSCode and initalize git in it

Follow these steps:

  • Open VScode Terminal by pressing " Ctrl + ` "
  • After making folder run the command "git init" on the VSCode Terminal

This command will initalize git in your local repository or folder at localhost

Step#5 Adding files to your local repository

Follow these steps:

  • Add some files like html, css or javascript files
  • Write some code in HTML file

We have added some files with code to our local repo now we can check it's affect

Step#6 Check Status/Affect after adding files

Follow these steps:

  • On VSCode terminal run command "git status"

This we show that some files are added to local repo...

Step#7 Add these files to git local server

Follow these steps:

  • On VSCode terminal run command "git add ." to add all the files that have been added or updated

We can add all files by running above command or we can add files individually by running "git add < filename > command on VSCode terminal"

Step#8 Commit the added files to git local server

Follow these steps:

  • On VSCode terminal run command "git commit -m "Write you message here" " to commit all the files that have been added

Now our files have been committed to git local server of our local repository

Step#9 Connecting local repository to remote repository we made on the GitHub server in Step#2

First of all check if our local repository is already connnect to remove server or not:

  • On VSCode terminal run command "git remote -v" to check if repo is remotely connected

This repository is not connect to any remote repository/server so we have to connect it to our remote repository/server we made at Step#2.

Step#10 Connecting local repository to remote repository we made on the GitHub server in Step#2

Copy the URL of the remote repository made on GitHub's remote server at Step#2

  • Open the repo we created and Copy it's URL as shown in figure

URL will be copied and we will now connect our local repository to remote server with this URL

Step#11 Connecting local repository to remote repository we made on the GitHub server in Step#2

Follow these steps :

  • We will make connection by running "git remote add origin < URL >"
  • Or in our case run: "git remote add origin https://github.com/ZaynAliShan/roll22_assign_gitweb.git"

URL will be copied and we will now connect our local repository to remote server with this URL

Step#12 Push changes that were committed in local repository to the Remote Repository to which we just connected to

Follow these steps :

  • On VSCode Terminal run: "git push -set--upstream origin master"

Now all the changes have been pushed and made to the local as well as the remote repository.

Git Branches

Step#1 Branch Create

Now we will create a git branch named as "feature" and will check if it's made or not

  • On the VSCode Terminal, run: "git branch < branchname >" and in our case run: "git branch feature" to craete a branch named as feature
  • On the VSCode Terminal, run: "git branch" to check what branches are there in our repo.

In our repo, we have made a branch named as feature

Step#2 Go to Branch "feature"

We can move from our master branch to feature branch by:

  • On the VSCode Terminal, run: "git checkout < branchname >" and in our case run: "git checkout feature" to move to branch, named as feature

Now we are in our "feature" branch

Step#3 Create and Move to other branch at same time

We can create a branch and move to that branch at the same time:

  • On the VSCode Terminal, run: "git checkout -b < branchname >" and in our case run: "git checkout -b feature2" to craete a branch named as feature2 and move to it at same time

Now we have craeted branched feature2 and moved to it

Step#4 Delete a branch

Follow these steps :

  • On the VSCode Terminal, run: "git branch -d < branchname >" and in our case run: "git branch -d feature2" to delete it
