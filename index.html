Git Fundamentals

Step#1 Git Repository

Follow these steps :

  • Log in your github account at any brower.
  • To make a git repository, do as shown in the picture:
  • Click on "+" sign and click "New repository"

This is how we can add a git repository to our account...

Step#2 Make a Git Repository

Follow these steps to make a git repository :

  • Enter repository name
  • Click on create repository button

We can either make it as public repository (will be visibel to everyone) or a private repository (will be visible to only you or any collaborators you add).

Step#4 Make a Project on your localhost/Personal Computer on VSCode and initalize git in it

Follow these steps:

  • Open VScode Terminal by pressing " Ctrl + ` "
  • After making folder run the command "git init" on the VSCode Terminal

This command will initalize git in your local repository or folder at localhost

Step#5 Adding files to your local repository

Follow these steps:

  • Add some files like html, css or javascript files
  • Write some code in HTML file

We have added some files with code to our local repo now we can check it's affect

Step#6 Check Status/Affect after adding files

Follow these steps:

  • On VSCode terminal run command "git status"

This we show that some files are added to local repo...

Step#7 Add these files to git local server

Follow these steps:

  • On VSCode terminal run command "git add ." to add all the files that have been added or updated

We can add all files by running above command or we can add files individually by running "git add < filename > command on VSCode terminal"
